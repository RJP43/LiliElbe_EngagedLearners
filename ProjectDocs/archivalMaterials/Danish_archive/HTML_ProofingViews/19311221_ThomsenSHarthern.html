Guide to Proofing

The following is a guide to our style coding for proofing of people, places, idiosyncratic spelling, and Editor/Translator comments. This color coding is intended to remind proofers to mouse over these particular parts of the text while proofing. Please mouse-over every person and place, with the Key ID styling, to verify the key ID matches the information available in the LEDA_Prosopography.xml.
